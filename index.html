📋 Setup Guide

How to connect your Notion data:

  1. Go to notion.so/my-integrations → Create a new integration → Copy the Internal Integration Token
  2. Open this HTML file → Find const CONFIG at top of the <script> → Paste your token in NOTION_TOKEN
  3. In Notion, go to your Daily Habits database → Share → Connect to your integration
  4. In Notion, go to your Di's Tasks database → Share → Connect to your integration
  5. Push to GitHub Pages — done!

⚠️ Note on CORS: Notion's API doesn't allow direct browser calls. You'll need a small proxy. The easiest free option: deploy a Cloudflare Worker or Vercel Edge Function that forwards requests and adds your token. See the proxyUrl in CONFIG.
